**Bytecode** é uma forma intermediária de código gerada pelo compilador Java ao transformar o código fonte (arquivos .java) em um formato que pode ser interpretado pela **Java Virtual Machine (JVM)**. Ele não é executável diretamente no hardware, mas é otimizado para ser interpretado pela JVM, permitindo a execução de códigos Java em diferentes plataformas sem necessidade de recompilação.
